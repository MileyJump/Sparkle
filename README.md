### Git Flow
- main  
	- 배포 가능 상태 유지 브랜치
	- 모든 개발 작업이 완료된 후 병합 
- develop
	 - QA 작업용 브랜치
	 - 개발 중인 최신 코드 포함
	 - 각 기능 작업이 완료된 후 병합
- feat 
	- 새로운 기능 개발을 위한 브랜치
- 각 브랜치별로 통일성과 작업 내용 확인을 위해 컨벤션 도입
	- prefix/#이슈번호-작업설명
	- feat/#5-channel-ui

## 사용 기술
iOS: UIKit, ReactorKit, RxSwift, RxCocoa
Architectur: Reactor
NetWork: Moya
Local DataBase: Realm
Design Pattern: Singlngton
Management: Git, Github, Figma
